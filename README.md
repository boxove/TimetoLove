# 时光予你

一个用 PHP 编写的个人纪念主页，包含首页、爱情记事页和 100 件事清单页。

## 功能概览

- 首页展示头像、昵称、签名、所在地、语录、导航入口和背景星空效果。
- 首页内置 APlayer 音乐播放器，支持歌词展示。
- 支持浅色/深色主题切换，并通过 `localStorage` 记住主题偏好。
- 支持展示支付宝、微信、QQ 等二维码入口。
- 爱情记事页展示双方生日倒计时、周年纪念日倒计时和重要事件已过去时间。
- 100 件事页面展示愿望清单、完成状态、完成数量和进度百分比。

## 项目结构

```text
.
├── index.php        # 首页
├── love.php         # 爱情记事 / 倒计时页面
├── love100.php      # 100 件事愿望清单页面
├── css/
│   ├── style.css    # 首页样式
│   ├── love.css     # 爱情记事与 100 件事页面样式
│   ├── js/          # jQuery、APlayer、clipboard 等前端脚本
│   ├── font/        # APlayer 字体资源
│   └── music/       # 本地音乐资源
└── img/             # 头像、图标、二维码、页面图片等资源
```

## 环境要求

- PHP 7.4 或更高版本。
- 一个静态资源可访问的 Web 服务环境，例如 Apache、Nginx。
- 浏览器需支持 JavaScript，倒计时、音乐播放器和主题切换依赖前端脚本运行。


常用页面：

- 首页：`/index.php`
- 爱情记事：`/love.php`
- 100 件事：`/love100.php`

## 内容配置

### 首页配置

打开 `index.php`，主要修改顶部配置区：

- `$site`：站点标题、关键词、描述、头像、昵称、签名、所在地、备案号等。
- `$navLinks`：首页导航链接。
- `$qrCodes`：二维码名称与图片路径。
- `$music`：歌曲名、歌手、音频地址、封面和歌词。

示例：

```php
$site = [
    'title' => '时光予你',
    'nickname' => '时光予你',
    'slogan' => '“你惊艳了我的时光，也温柔了我的岁月”',
];
```

### 爱情记事配置

打开 `love.php`，主要修改顶部配置区：

- `$site`：页面标题、描述、图标、Logo 和介绍文案。
- `$people`：双方姓名、农历生日和头像图片。
- `$anniversary`：周年纪念日的农历日期和图片。
- `$pastEvents`：已经发生的重要事件标题、日期和图片。

日期说明：

- `$people` 与 `$anniversary` 使用农历日期字段：`lunarYear`、`lunarMonth`、`lunarDay`。
- `$pastEvents` 使用公历日期字符串，格式为 `YYYY-MM-DD`。

### 100 件事配置

打开 `love100.php`，修改 `$loveThings` 数组：

```php
['text' => '一起看日出', 'done' => false],
['text' => '一起包饺子', 'done' => true],
```

字段说明：

- `text`：愿望内容。
- `done`：是否完成，`true` 表示已完成，`false` 表示待完成。

页面会自动统计总数、已完成数量、待完成数量和完成百分比。

## 图片与资源

- 站点图标和头像默认使用 `img/ico.png`。
- 首页音乐封面默认使用 `img/tm.png`。
- 二维码、人物头像、事件图片等资源建议放在 `img/` 目录下。
- 修改图片文件名后，需要同步更新对应 PHP 配置中的路径。

注意：如果页面中配置的图片路径不存在，浏览器会显示图片加载失败。部署前建议检查 `img/` 目录中的资源是否完整。

## 部署说明

1. 将项目文件上传到服务器站点根目录。
2. 确认服务器已启用 PHP。
3. 确认 `index.php` 被设置为默认首页，或直接访问 `/index.php`。
4. 检查 `css/`、`img/` 等静态资源目录是否可公开访问。
5. 如需备案信息，修改 `index.php` 中的 `icp` 和 `icp_url`。

## 常见问题

### 音乐无法播放

- 检查 `$music['url']` 是否可访问。
- 如果使用外链音频，确认目标地址支持浏览器直接播放且没有防盗链限制。
- 可以将音乐文件放入 `css/music/` 后改为本地路径。

### 二维码不显示

- 检查 `$qrCodes` 中配置的图片路径是否真实存在。
- 图片路径建议使用相对路径，例如 `./img/wechat.svg`。

### 倒计时不准确

- 生日和周年纪念日按农历计算，确认 `lunarYear`、`lunarMonth`、`lunarDay` 是否填写正确。
- 已发生事件按公历日期计算，确认日期格式为 `YYYY-MM-DD`。

### 深色模式没有记住

- 深色模式依赖浏览器 `localStorage`。
- 隐私模式、浏览器清理缓存或禁用本地存储时，主题偏好可能无法保存。

## 维护建议

- 修改配置前先备份当前文件。
- 上传新图片后，保持文件名简短且不要包含空格。
- 部署后使用手机和电脑分别检查显示效果。
- 如更换音乐、图片或脚本外链，优先使用 HTTPS 地址。
